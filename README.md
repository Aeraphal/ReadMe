## 🏆 Projets Phares

### 🎳 [Projet Majeur : Jeu de Quilles en Réalité Augmentée](./5eme-annee/projet-majeur-ar-quilles/)
**Projet de fin d'année - 5ème année**  
Développement d'un jeu de quilles en réalité augmentée avec tracking en temps réel d'une balle de tennis dans un espace 3D. Combinaison Unity + Python pour la détection et le suivi.

`Unity` `Python` `AR` `Computer Vision` `Tracking 3D`

### 🥽 [Projet VR : Jeu avec Caméra Libre & Physique](./5eme-annee/projet-vr-unity/)
**5ème année**  
Création d'un jeu avec caméra libre et interactions physiques. Implémenté sur mobile et casque VR avec contrôles adaptés.

`Unity` `VR` `Mobile` `Physique` `Interactions 3D`

### 👁️ [Calibrage & Reconnaissance Braille](./4eme-annee/calibrage-braille/)
**4ème année**  
Calibrage de l'acquisition numérique puis lecture d'image en braille grâce à la reconnaissance de motifs suivant l'alphabet français du braille.

`Python` `Computer Vision` `Calibrage` `Pattern Recognition`

### 🌐 [Site E-commerce TLW](./3eme-annee/projet-tlw-web/)
**3ème année**  
Site web complet de vente en ligne avec interface moderne et interactions dynamiques.

`HTML` `CSS` `JavaScript` `Web Design`
